# number2letter

Convert numbers to words in multiple languages. Written in TypeScript, ships with type declarations.

| Language | Code | 42 becomes |
|---|---|---|
| English (default) | `en` | forty-two |
| French | `fr` | quarante-deux |
| Khmer | `km` | бһҹбҹӮбһҹбһ·бһ”бһ–бһёбһҡ |

## Install

```sh
npm install number2letter
```

## Usage

```ts
import { numberToWords, languages } from "number2letter";

numberToWords(42);          // "forty-two" (English is the default)
numberToWords(42, "fr");    // "quarante-deux"
numberToWords(42, "km");    // "бһҹбҹӮбһҹбһ·бһ”бһ–бһёбһҡ"

numberToWords(1234, "km");  // "бһҳбһҪбһҷбһ–бһ¶бһ“бҹӢбһ–бһёбһҡбһҡбһҷбһҹбһ¶бһҳбһҹбһ·бһ”бһ”бһҪбһ“"
numberToWords(-7);          // "minus seven"

languages;                  // ["en", "fr", "km"]
```

## API

### `numberToWords(num: number, lang?: Language): string`

Converts an integer to words in the given language (default `"en"`). Works for any safe integer, positive or negative (up to Вұ9,007,199,254,740,991). Throws a `TypeError` for decimals, `NaN`, `Infinity`, or numbers beyond the safe integer range, and a `RangeError` for unsupported language codes.

### `languages: Language[]`

The list of supported language codes.

## Adding a language

1. Create `src/locales/<code>.ts` exporting a `LocaleConverter` вҖ” a function that takes a safe integer and returns the words.
2. Register it in the `LOCALES` object in `src/index.ts`.
3. Add a `test/<code>.test.ts` with expected outputs.

The `Language` type, the `languages` list, and validation all update automatically from the `LOCALES` object.

## Development

Requires Node.js 24+ (runs TypeScript directly, no build step needed for tests).

```sh
npm install          # install dev dependencies
npm test             # run the test suite
npm run typecheck    # type-check without emitting files
npm run build        # compile src/ to dist/
node example.ts      # run the example
```
